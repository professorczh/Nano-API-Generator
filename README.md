================================================================================

多模态 AI 无限画布创作终端 | 自由探索、精准标注、隐私至上

================================================================================

## 🌐 快速开始 (在线免安装)

无需配置复杂的后端环境，直接访问在线版本即可开始创作：

[👉 立即访问 Nano Generator](https://nano-api-generator.vercel.app/)

> **💡 使用提示：**
> 1. 点击左下角 **设置 (Settings)**。
> 2. 填入您的 API Key（数据仅保存在您的浏览器本地）。
> 3. 即可开始在 10000x10000 的大画布上自由创作。

## ✨ 核心特性

### 🎨 空间化交互 (Infinite Canvas)

| 功能 | 说明 |
|------|------|
| 无限画布 | 提供 10,000x10,000 像素的超大创作空间，打破聊天框的限制 |
| 精准缩放 | 针对设计场景优化的缩放算法，支持以鼠标指针为中心进行丝滑无损缩放 |
| 小地图导航 | 内置实时雷达视图，在大画布中永远不会迷失方向 |

### 📍 视觉对齐 (PIN 系统)

| 功能 | 说明 |
|------|------|
| 视觉引导 | 通过 **Ctrl + 点击** 图片，可生成精准的坐标标记 (PIN) |
| 空间感知 | 系统自动将视觉标记转化为空间提示词，让 AI 理解"这里"、"那边"的具体含义 |

### 🤖 多供应商兼容

| 功能 | 说明 |
|------|------|
| 模型无关 | 支持 Google Gemini 原生接口、OpenAI 兼容格式以及 Claude 等主流 AI 供应商 |
| 并发任务 | 支持多个模型任务并行生成，每个节点独立计时，效率翻倍 |

## 🕹️ 交互快捷键

| 功能 | 快捷键 / 动作 |
|------|---------------|

## 🚀 本地开发与部署

如果您希望在本地环境中运行，请参考以下指令：

### 1. 克隆仓库

```bash
git clone https://github.com/professorczh/Nano-API-Generator.git
cd Nano-API-Generator
```

### 2. 本地启动 (Node.js)

```bash
# 安装依赖 (如适用)
npm install

# 启动服务
node server.js
```

### 3. 使用 Docker 构建镜像

```bash
# 构建镜像
docker build -t nano-generator .

# 运行容器 (支持生成的资源自动同步到本地 DL 文件夹)
docker run -d -p 8000:8000 -v "$(pwd)/DL:/app/DL" --name nano-generator nano-generator
```
